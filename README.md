# ワイン品質分析（Vinho Verde）--- 統計検定と重回帰分析

本リポジトリは、ポルトガル産 Vinho Verde
ワイン（赤・白）品質データを用いたデータサイエンス実践ノートです。
統計的仮説検定および重回帰分析（OLS）を用いて、ワインの化学特性と品質評価の関係を分析しています。

------------------------------------------------------------------------

## 1. 目的

本分析では以下を検証します。

-   赤ワインと白ワインで平均品質に差があるか（Welchのt検定）
-   どの化学特性が品質に影響しているか
-   重回帰モデル（OLS）でどの程度品質を説明・予測できるか

------------------------------------------------------------------------

## 2. データセット

UCI Machine Learning Repository の公式データを使用しています。

-   winequality-red.csv
-   winequality-white.csv

各データには以下が含まれます。

-   物理化学的特性（酸度、糖度、アルコール度数など）
-   品質スコア（quality）

wine_type 列（red / white）を追加し、両データを結合して分析しています。

------------------------------------------------------------------------

## 3. 分析内容

### 仮説検定

-   Welchのt検定（分散不等）
-   95%信頼区間の算出
-   手計算によるWelch検定の実装

### 重回帰分析（OLS）

-   quality \~ C(wine_type) + 11個の化学特性
-   回帰係数の解釈
-   多重共線性の考察
-   density除外モデルによる頑健性確認

### 予測

-   新規ワインサンプルの品質予測
-   95%信頼区間および予測区間の算出

------------------------------------------------------------------------

## 4. 主な知見

-   赤と白の品質平均には統計的有意差がある
-   volatile acidity は品質と負の関連
-   alcohol は品質と正の関連
-   線形モデルのみでは品質を完全には説明できない（R² ≈ 0.30）
